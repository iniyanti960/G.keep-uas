# Google Keep CLI

A simple Google Keep clone built using Java, Maven, and SQLite.

## Features

- Add Note
- View Notes
- Edit Note
- Delete Note
- Archive Note
- Pin Note
- Search Note

## Technologies

- Java 21
- Maven
- SQLite
- JUnit
- Mockito

## How to Run

### Clone Repository

```bash
git clone https://github.com/iniyanti960/G.keep-uas.git
```

### Open Project

```bash
cd G.keep-uas
```

### Compile Project

```bash
mvn compile
```

### Run Program

Run `Main.java` from VS Code or IntelliJ.

Or using Maven:

```bash
mvn exec:java -Dexec.mainClass="com.keep.Main"
```
